Kiosk Device Provisioning

Scan during Android setup to install app and set Device Owner automatically

Steps:
  1. Factory reset the tablet
  2. On the first "Welcome" screen, tap 6 times quickly
  3. Connect to WiFi when prompted
  4. Scan this QR code
  5. Wait for app to download and install (~2 min)
  6. Device Owner will be set automatically

Package: ca.looknet.familykiosk
